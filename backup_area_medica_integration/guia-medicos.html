🔄 Fluxo Completo de Teste

1

Paciente Faz Lance

Acesse e clique "Fazer Lance" em qualquer especialidade.

💡 Tip: Use valores como R$ 200 para Cardiologia ou R$ 150 para Pediatria
2

Sistema Salva no localStorage

O lance é automaticamente salvo e o paciente é redirecionado para página de aguardo.

🔍 Debug: Use F12 → Console para ver logs detalhados do processo
3

Médico Vê Lance

Acesse e veja o lance na seção "Lances Ativos".

⚡ Auto-refresh: A página atualiza a cada 10 segundos automaticamente
4

Médico Aceita

Clique "Atender" no lance → Confirma → Redireciona automaticamente para videoconsulta.

🔧 Tecnologias Utilizadas

  • React.js + TypeScript
  • Node.js + Express

  • Tailwind CSS
  • localStorage para demo

  • Load time < 3 segundos

  • WebRTC latency < 100ms
  • Mobile responsive
  • Cross-browser compatible
  • HTTPS ready
