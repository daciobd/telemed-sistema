Como Usar no Hostinger

  1. Acesse o Editor: Entre no Hostinger Website Builder
  2. Selecione o Botão: Clique no botão que deseja configurar
  3. Abra Configurações: Procure por "Link" ou "URL" nas configurações
  4. Cole o Link: Use os botões "Copiar Link" acima para copiar a URL
  5. Configure Abertura: Selecione "Abrir em nova aba" para melhor experiência
  6. Salve e Publique: Salve as alterações e publique o site
