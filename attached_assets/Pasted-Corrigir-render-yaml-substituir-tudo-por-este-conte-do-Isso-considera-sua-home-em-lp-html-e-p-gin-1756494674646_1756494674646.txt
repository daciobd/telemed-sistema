2) Garanta que o arquivo da rota existe

O erro “Permission denied” apareceu porque você tentou executar um arquivo HTML no shell (public_live/demo-responsivo/index.html). Para checar se existe, use ls/sed, não execute.

# Verificar
ls -la public_live/demo-responsivo || true

# Se ainda não existir, crie a pasta e copie a página
mkdir -p public_live/demo-responsivo

# Tente copiar de onde você tem a versão boa do demo:
# (ajuste a origem conforme seu repo; alguns caminhos comuns:)
if [ -f public/preview/demo-responsivo.html ]; then
  cp -f public/preview/demo-responsivo.html public_live/demo-responsivo/index.html
elif [ -f public_live/preview/demo-responsivo.html ]; then
  cp -f public_live/preview/demo-responsivo.html public_live/demo-responsivo/index.html
fi

# confira:
sed -n '1,80p' public_live/demo-responsivo/index.html
