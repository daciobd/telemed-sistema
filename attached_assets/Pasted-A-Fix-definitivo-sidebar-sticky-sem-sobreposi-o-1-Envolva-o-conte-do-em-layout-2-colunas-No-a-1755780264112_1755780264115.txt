A) Fix definitivo (sidebar sticky, sem sobreposição)
1) Envolva o conteúdo em layout 2 colunas

No arquivo da página /pacientes (ou componente), agrupe os painéis “Mais Páginas / Páginas Adicionais” num <aside> e deixe o resto no <section>:

<main class="page patients">
  <div class="page-header">…</div>

  <div class="layout-2col">
    <section class="content">
      <!-- TODO: blocos “Meus Pacientes”, cards, listas… -->
      …conteúdo principal…
    </section>

    <aside class="pages-sidebar">
      <div class="card">… Páginas Adicionais …</div>
      <div class="card">… Mais Páginas …</div>
    </aside>
  </div>
</main>


Não precisa reescrever os cards; só mova esses dois blocos para dentro do <aside>.

2) CSS (cola e pronto)

Crie client/src/styles/patients.sidebar.css e importe no bundle (ou cole no seu CSS global):

/* layout em duas colunas */
.page.patients .layout-2col{
  display:grid;
  grid-template-columns: 1fr 340px;
  gap:24px;
  align-items:start;
}

/* sidebar fixa enquanto rola (sem cobrir conteúdo) */
.page.patients .pages-sidebar{
  position: sticky;
  top: calc(var(--header-h, 72px) + 12px);
  z-index: 10; /* abaixo do header */
}

/* visual de cartão */
.page.patients .card{
  background: var(--card, #1f2937);
  border: 1px solid var(--border, #2b3645);
  border-radius: 12px;
  padding: 12px;
  box-shadow: 0 10px 24px rgba(0,0,0,.15);
}

/* responsivo: 1 coluna, sidebar sobe para cima do conteúdo */
@media (max-width:1100px){
  .page.patients .layout-2col{ grid-template-columns: 1fr; }
  .page.patients .pages-sidebar{ position: static; order: -1; margin-bottom: 12px; }
}

/* cards com avatar (evita texto invadir imagem) */
.patient-card{ display:flex; align-items:center; gap:12px; }
.patient-card img{ width:48px; height:48px; border-radius:12px; object-fit:cover; flex:0 0 48px; }
.patient-card .info{ min-width:0; }
.patient-card .name{ overflow:hidden; white-space:nowrap; text-overflow:ellipsis; }


Se o header é fixo, exponha a altura em --header-h (opcional). Ex.: no load do app:

const h = document.querySelector('header')?.offsetHeight || 72;
document.documentElement.style.setProperty('--header-h', `${h}px`);
